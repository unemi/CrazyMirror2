これはなに？

フルHDカメラ と macOS 11 または上位の OS の上で動く役立たずなアプリです。 Intel と Apple Silicon どちらの CPU でも動作しますが、 Intel製 GPU では、複雑な効果を滑らかに実行するには力不足かもしれません。

元祖版 は 2005年 (平成17年) に iSight カメラと QuickTime API を使って開発されました。 その後、新しい OS では動かなくなってしまったのですが、プログラムを一から作り直し、ついに復活しました。 Metal シェーダ言語のコードを含んでいます。

使い方

仕事で疲れを感じたらこのソフトを立ち上げてください。 ゆっくり首を回して、肩を上下に動かし、軽くマバタキをしてから、 ３回深呼吸をして下さい。

まだ疲れを感じる場合は、 ウィンドウを上の方にあるツールバーの中の「こうか」のボタンをクリックして、別のモードを試してみましょう。

それで気分がどうなるかっていわれても何の保証もできませんけど...

履歴

  • 令和5年5月16日: バージョン 1.2 を公開。 「ひょうじ」メニューで、ウインドウの大きさをいくつかの固定サイズから選べるようになりました。 全画面モードでの操作ができるよう、コンテクストメニューを追加しました。右クリック（または Control+クリック) で表示されます。 ビデオ録画中に赤い点が点滅するようにしました。
  • 令和5年5月15日: バージョン 1.1 を公開。 2つ以上のカメラがつながっている場合に、使用するカメラを選択できます。 写真とビデオを撮影して 写真 ライブラリに保存できます。
  • 令和5年5月11日: バージョン 1.0 を公開。

バグ

Intel CPU のマシンでは、ビデオ録画が動作せず、アプリがクラッシュすることがあります。
